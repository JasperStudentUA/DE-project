# DE-project
Data Engineering project

Imortant commands;

pip install -r requirements.txt

Get the newest files/ changes:
git pull origin main

Push your changes to Github so the whole team can see (copy the whole and paste in terminal)
git add .
git commit -m "comment"
git push

Interactive Plotly versions are stored as HTML files in report/figures/xgb_v2/.
For GitHub preview, static PNG versions are provided in the same folder.
